LA TIENDA DE AUTUMN

La tienda oficial de Autumn64

Todas estas aplicaciones y recursos son totalmente libres y de código abierto, y están disponibles para Windows y GNU/Linux.

No tienes que pagar por ninguna de estas cosas. Aún así, ¡gracias por todas las donaciones!

RaccoonLock

Un gestor de contraseñas simple, poderoso, libre y de código abierto.

RaccoonLock es un gestor de contraseñas multiplataforma escrito en Electron y C, diseñado para proveer un gestor de contraseñas ligero, sencillo y seguro para usuarios que quieren guardar sus contraseñas de forma segura sin conexión.

PyTDownloader

Un descargador de videos de YouTube portable, libre y de código abierto.

PyTDownloader es un descargador de videos de YouTube escrito en Python, que provee una interfaz simple y moderna para descargar cualquier video de YouTube en los formatos .mp4 y .mp3.

Keyes

Un bot para Discord libre, ético y de código abierto que respeta la privacidad de los usuarios.

Keyes es un bot para Discord ético, libre y de código abierto, orientado a la privacidad y basado en el código de MOONSTAR, inspirado en la saga de videojuegos Halo.

Discoabber

Un puente entre Discord y XMPP/Jabber, libre y de código abierto.

Discoabber es un puente para Discord y XMPP/Jabber descentralizado y escrito en Python.

SMARTSTR.H

Una librería de Strings para C libre, de código abierto, simple, segura en memoria y ligera.

SMARTSTR.H es una librería para C que provee una interfaz para trabajar con cadenas de texto de manera similar a los lenguajes de programación de alto nivel más comunes. SMARTSTR.H significa "SMART Strings Header file".

hyprmonitor

Un gestor de monitores para Hyprland basado en el terminal, simple, libre y de código abierto.

hyprmonitor (escrito en minúsculas) es un frontend para hyprctl simple y de terminal escrito en C, diseñado para proveer un gestor de monitores fácil de usar para Hyprland.

Wiki Guía de LaTeX
